Идея данного компонента появилась после прочтения статьи про математику CSS-шлюзов.
Чтобы не считать различные адаптивные значения вручную, данный алгоритм был запрограммирован.

Изменение размера - это простая линейная функция (см. ниже).

Она записывается в виде y = mx + b, где:

  • y — наш размер (вертикальная ось)
  • x — ширина области просмотра, в пикселях (горизонтальная ось)
  • m — крутизна наклона функции («сколько пикселей прибавляется к размеру на каждый пиксель увеличения ширины окна?»),
  • и b — размер шрифта до того, как к нему добавится какое-либо относительное значение, связанное с размером окна.
