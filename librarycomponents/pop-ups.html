Ссылка на модальное окно создается следующим образом:

A link to the fashion window is created as follows:

<a href="#privacyPolicy" class="open-modal"> <span data-lang="ru">Политика конфиденциальности</span> <span data-lang="en">Privacy policy</span> </a>

-В атрибуте href указывается id модального окна
-Для ссылок обязательный класс open-modal
-Внутри ссылки указывается текст (если сайт мультиязычный, то используется элемент span c атрибутом data-lang=ru и data-lang=en, в зависимости от языков на сайте)

-The href attribute specifies the id of the modal window
-For references, a compulsory class open-modal
-Inside the link, the text is specified (if the site is multilanguage, then a span element is used with the attribute data-lang=ru and data-lang=en, depending on the languages on the site)

Само модальное окно определяется в самом конце страницы следующим образом:

The modal window itself is defined at the very end of the page as follows:

<div id="privacyPolicy" class="modal"> <div class="modal__body"> <span class="modal__close">&times;</span> <div class="modal__content"> ... </div> </div> </div>

-Для модальных окон обязательный класс modal
-Тело модального окна необходимо определять классом modal__body
-Крестик, который закрывает окно, определяется классом modal__close (также окно можно закрыть нажав клавишу Esc или по области вне модального окна)
-Весь основной контент находится в блоке с классом modal__content

-For modal windows, the required class modal
-The modal window's body must be defined by the modal__body class
-The shortcut that closes the window is defined by the modal__close class (You can also close the window by pressing the Esc key or the area outside the modal window)
-The whole main content is in the block with the class modal__content
